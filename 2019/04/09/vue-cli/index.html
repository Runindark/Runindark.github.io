vue-cli 安装

安装node.js

过程略 使用

node -v

可以查看

安装cnpm

利用淘宝加速镜像

npm install -g cnpm –registry=http://registry.npm.taobao.org

安装vue-cli手脚架

npm install -g vue-cli

初始化vue-cli项目

利用 webpack 打包

vue init webpack myvue

根据提示完成代码运行

cd myvue

cnpm install

cnpm run dev
